# Fofan Montage / Fofan Flow

Готовый комплект для репозитория `4fan777/Fofan-Flow`:

- `miniapp/` — Telegram Mini App для GitHub Pages.
- `worker/` — Cloudflare Worker + D1: Telegram-вход, тарифы, циклы действий, уведомления и синхронизация лицензий.
- `bot/` — aiogram-бот с кнопкой Mini App, deep-link заказами и синхронизацией подписок.
- `plugin/` — исходники Fofan Flow с учётом действий и запусков.
- `.github/workflows/pages.yml` — публикация `miniapp/` на GitHub Pages.

## Быстрый запуск

1. Создайте приватный репозиторий или загрузите содержимое в `4fan777/Fofan-Flow`.
2. В `miniapp/config.js` укажите URL Worker и username бота.
3. Создайте D1 и Worker по инструкции `docs/DEPLOY_RU.md`.
4. Заполните `.env` бота по `bot/.env.example`.
5. В `plugin/Extension/FofanFlow/client/usage-config.js` укажите тот же Worker URL.
6. После проверки включите GitHub Pages: **Settings → Pages → GitHub Actions**.

## Важное ограничение по оплате

Владелец проекта сообщил, что ему 17 лет и статус продавца ещё не оформлен. Поэтому публичную оферту и реальный приём оплаты нельзя запускать от имени несуществующего ИП/самозанятого/ООО. До оформления через законного совершеннолетнего продавца платёжный сценарий следует считать тестовым.

Секреты (`BOT_TOKEN`, токены администраторов, `DATA_KEY`) не коммитить.
